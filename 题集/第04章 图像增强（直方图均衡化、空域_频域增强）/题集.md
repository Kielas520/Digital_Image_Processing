# 第04章 图像增强 测试题集

## 第一次测试 (2026-06-15)

**Q1: 中值滤波会消除连续个数小于窗口宽度一半的离散脉冲，并削平三角形信号的顶部。因此，以下哪类图像最不适合使用中值滤波？**
- 选项：A. 含有大面积灰度平缓的图像 | B. 含有孤立椒盐噪声点的图像 | C. 包含大量细线、尖角等精致细节的图像 | D. 由于摄像机抖动产生运动模糊的图像
- 用户答案：C
- 答案：C
- 解析：中值滤波的排序统计特性决定了它会将细线、孤立尖角误判为噪声而强行削平（剔除）。因此，包含丰富精细点、线、尖角的细节图像学绝不适合中值滤波，会造成严重的细节损伤。

**Q2: 在常用的空域微分锐化算子中，属于各向同性（即对任何方向的边缘检测敏感度相同）的二阶微分算子是：______。**
- 选项：A. 罗伯茨（Roberts）算子 | B. 普雷维特（Prewitt）算子 | C. 索贝尔（Sobel）算子 | D. 拉普拉斯（Laplacian）算子
- 用户答案：D
- 答案：D
- 解析：罗伯茨、Prewitt、Sobel 都是一阶微分（梯度）算子，具有方向性（如水平或垂直模板）。只有拉普拉斯算子是旋转不变、各向同性的二阶微分算子。

**Q3: 在空域卷积滤波中，关于平滑（低通）模板与纯边缘检测（高通）模板的系数之和，下列说法正确的是：______。**
- 选项：A. 平滑模板系数之和为1；纯边缘检测锐化模板系数之和为0 | B. 平滑模板系数之和为0；纯边缘检测锐化模板系数之和为1 | C. 两者的系数之和都必须恒等于 0 | D. 两者的系数之和都必须恒等于 1
- 用户答案：A
- 答案：A
- 解析：平滑模板所有系数为正且和为1（保证图像均值亮度不变）；纯边缘检测模板有正有负，在灰度平坦区（无边缘）卷积应输出0，因此所有系数之和恒等于0（如拉普拉斯模板 $4 - 1 - 1 - 1 - 1 = 0$）。

**Q4: 为了实现拉普拉斯图像增强（即锐化且保留原图背景亮度），当采用 4-邻域 差分时，其对应的图像拉普拉斯增强卷积模板的中心系数应为______。**
- 选项：A. 中心系数为 4；周围为 -1 | B. 中心系数为 5；周围为 -1 | C. 中心系数为 8；周围为 -1 | D. 中心系数为 9；周围为 -1
- 用户答案：B
- 答案：B
- 解析：纯拉普拉斯检测模板中心为 4，周围为 -1，其元素和为 0，卷积后背景全变黑。为了保留原背景，必须用原图减去拉普拉斯算子：$g = f - \nabla^2 f = 5f - [f(x+1,y)+\dots]$。因此增强模板的中心系数为 5，周围为 -1，此时系数和为 1，确保了处理后图像的平均亮度保持守恒。

## 15分必考大题：直方图均衡化与规定化计算（2026-06-15 归档）

**题目：一幅 64×64 大小的8级灰度图像，原直方图与规定直方图数据见 cache/question.md：**
1. **写出其直方图均衡化后的灰度映射关系。**
2. **写出其用单映射最小差值法求得的直方图规定化最终灰度映射表。**

- **答案与推导：**

### 1) 直方图均衡化映射表：
- **第一步：计算累计概率（CDF） $S_k$**
  - $S_0 = 0.19$
  - $S_1 = 0.19 + 0.25 = 0.44$
  - $S_2 = 0.44 + 0.21 = 0.65$
  - $S_3 = 0.65 + 0.16 = 0.81$
  - $S_4 = 0.81 + 0.08 = 0.89$
  - $S_5 = 0.89 + 0.06 = 0.95$
  - $S_6 = 0.95 + 0.03 = 0.98$
  - $S_7 = 0.98 + 0.02 = 1.00$
- **第二步：根据公式 $s_k = \text{round}[7 \times S_k]$ 映射取整**
  - $s_0 = \text{round}[7 \times 0.19] = \text{round}[1.33] = \mathbf{1}$
  - $s_1 = \text{round}[7 \times 0.44] = \text{round}[3.08] = \mathbf{3}$
  - $s_2 = \text{round}[7 \times 0.65] = \text{round}[4.55] = \mathbf{5}$
  - $s_3 = \text{round}[7 \times 0.81] = \text{round}[5.67] = \mathbf{6}$
  - $s_4 = \text{round}[7 \times 0.89] = \text{round}[6.23] = \mathbf{6}$
  - $s_5 = \text{round}[7 \times 0.95] = \text{round}[6.65] = \mathbf{7}$
  - $s_6 = \text{round}[7 \times 0.98] = \text{round}[6.86] = \mathbf{7}$
  - $s_7 = \text{round}[7 \times 1.00] = \text{round}[7.00] = \mathbf{7}$

- **最终均衡化映射表**：
  $$ 0 \rightarrow 1, \quad 1 \rightarrow 3, \quad 2 \rightarrow 5, \quad 3 \rightarrow 6, \quad 4 \rightarrow 6, \quad 5 \rightarrow 7, \quad 6 \rightarrow 7, \quad 7 \rightarrow 7 $$

### 2) 直方图规定化映射表（单映射最小差值法）：
- **第一步：计算规定直方图的累计分布概率 $V_q$ 与均衡化映射值 $v_q = \text{round}[7 \times V_q]$**
  - $V_0 = V_1 = V_2 = V_3 = 0.00 \rightarrow v_0 = v_1 = v_2 = v_3 = \mathbf{0}$
  - $V_4 = 0.20 \rightarrow v_4 = \text{round}[7 \times 0.20] = \text{round}[1.40] = \mathbf{1}$
  - $V_5 = 0.20 + 0.30 = 0.50 \rightarrow v_5 = \text{round}[7 \times 0.50] = \text{round}[3.50] = \mathbf{4}$
  - $V_6 = 0.50 + 0.30 = 0.80 \rightarrow v_6 = \text{round}[7 \times 0.80] = \text{round}[5.60] = \mathbf{6}$
  - $V_7 = 0.80 + 0.20 = 1.00 \rightarrow v_7 = \text{round}[7 \times 1.00] = \text{round}[7.00] = \mathbf{7}$

- **第二步：单映射匹配（寻找 $|s_k - v_q|$ 最小的 $q$）**
  - 对于 $r_0 \rightarrow s_0 = 1$：最接近的 $v_q$ 为 $v_4 = 1$，映射到目标灰度级 **$4$**。
  - 对于 $r_1 \rightarrow s_1 = 3$：最接近的 $v_q$ 为 $v_5 = 4$（差值1），映射到目标灰度级 **$5$**。
  - 对于 $r_2 \rightarrow s_2 = 5$：在 $v_5=4$ 和 $v_6=6$ 之间（差值均为1），向高灰度级规整，映射到目标灰度级 **$6$**。
  - 对于 $r_3 \rightarrow s_3 = 6$：最接近的 $v_q$ 为 $v_6 = 6$，映射到目标灰度级 **$6$**。
  - 对于 $r_4 \rightarrow s_4 = 6$：最接近的 $v_q$ 为 $v_6 = 6$，映射到目标灰度级 **$6$**。
  - 对于 $r_5 \rightarrow s_5 = 7$：最接近的 $v_q$ 为 $v_7 = 7$，映射到目标灰度级 **$7$**。
  - 对于 $r_6 \rightarrow s_6 = 7$：最接近的 $v_q$ 为 $v_7 = 7$，映射到目标灰度级 **$7$**。
  - 对于 $r_7 \rightarrow s_7 = 7$：最接近的 $v_q$ 为 $v_7 = 7$，映射到目标灰度级 **$7$**。

- **最终规定化直接映射表 $r_k \rightarrow z_q$**：
  $$ 0 \rightarrow 4, \quad 1 \rightarrow 5, \quad 2 \rightarrow 6, \quad 3 \rightarrow 6, \quad 4 \rightarrow 6, \quad 5 \rightarrow 7, \quad 6 \rightarrow 7, \quad 7 \rightarrow 7 $$
