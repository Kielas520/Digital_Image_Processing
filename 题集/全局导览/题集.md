# 全局导览 测试题集

> 记录了历次“全局导览”模式下的所有测试题目及解析，供后续全盘复习使用。

## 第一次测试 (2026-06-14)

**Q1: 一幅灰度均匀分布的图像，灰度范围在[0, 255]（256级），则该图像的信息量（熵）为？**
- 选项：A. 0 bit / B. 8 bit / C. 16 bit / D. 256 bit
- 答案：B
- 解析：公式 $H = \log_2 G$。256级均匀分布，$\log_2 256 = 8 \text{ bit}$。

**Q2: 傅立叶变换得到的频谱中，低频系数对应于图像的什么部分？**
- 选项：A. 物体边缘 / B. 噪声点 / C. 变化平缓部分 / D. 变化剧烈部分
- 答案：C
- 解析：低频对应平滑渐变的背景，高频对应边缘和噪声。

**Q3: 图像压缩是建立在三种数据冗余的基础上，以下哪项不属于这三种数据冗余？**
- 选项：A. 像素间冗余 / B. 空间域冗余 / C. 编码冗余 / D. 心理视觉冗余
- 答案：B
- 解析：正确术语为像素间冗余、编码冗余、心理视觉冗余。“空间域冗余”为干扰项。

**Q4: 下列关于中值滤波的说法，错误的是？**
- 选项：A. 对椒盐噪声抑制效果明显 / B. 是一种线性滤波方法 / C. 不影响阶跃、斜坡信号 / D. 不适合细节丰富的图像
- 答案：B
- 解析：中值滤波基于排序统计，属于非线性滤波器。

## 第二次测试 (2026-06-14)

**Q1: 存储一幅大小为 1024×1024，256个灰度级的图像，需要多大的存储空间？**
- 选项：A. 1024 Byte / B. 512 KB / C. 1 MB / D. 8 MB
- 答案：C
- 解析：$b = 1024 \times 1024 \times \log_2 256 \text{ bit} = 1048576 \text{ Byte} = 1024 \text{ KB} = 1 \text{ MB}$。

**Q2: 在空间域滤波中，以下哪种模板特征通常对应于平滑（低通）滤波器？**
- 选项：A. 模板系数有正有负，且和为0 / B. 模板中心系数较大，周围系数为负 / C. 模板系数全为正，且和为1 / D. 模板左列为负，右列为正
- 答案：C
- 解析：平滑模板要求系数全为正且和为1（保持图像亮度不变）。A和B为锐化高通模板，D为梯度算子。

**Q3: 以下编码方法中，属于有损压缩的是？**
- 选项：A. 霍夫曼(Huffman)编码 / B. 行程编码(RLE) / C. 算术编码 / D. 基于DCT变换的JPEG标准压缩
- 答案：D
- 解析：霍夫曼、香农-费诺、算术编码、行程编码均为无损压缩。JPEG因包含量化步骤，属于有损压缩。

**Q4: JPEG图像压缩系统中对8×8图像子块做正交变换，采用的是哪种变换？**
- 选项：A. 离散傅里叶变换(DFT) / B. 离散余弦变换(DCT) / C. 沃尔什-哈达玛变换(WHT) / D. 小波变换
- 答案：B
- 解析：JPEG采用DCT（离散余弦变换），因其只有实数运算且具有极好的能量集中特性。
