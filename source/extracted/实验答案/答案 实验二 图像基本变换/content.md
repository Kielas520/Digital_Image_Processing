| 函数名 | 功能描述 |
| --- | --- |
| imresize(I,2) | 图像尺寸的缩放 |
| fliplr(),flipud | 图像水平镜像、垂直镜像 |
| imrotate() | 图像旋转 |
| permute(I,[2,1,3]) 或者使用旋转和镜像实现 | 图像转置 |
| fft2(),fftshift() | 图像的离散傅里叶变换，图像频谱中心化 |
| dct2() | 图像的离散余弦变换 |
