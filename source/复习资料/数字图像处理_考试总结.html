一、基础概念

知识点

图像数字化过程：
模拟图像 → 采样（空间坐标离散化）→ 量化（幅值离散化）→ 编码 → 数字图像

图像存储量公式：

b=M×N×log⁡2G(bit)b = M \times N \times \log_2 G \quad \text{(bit)} b=M×N×log2​G(bit)

  • M×N 为图像尺寸，G 为灰度级数
  • 换算：÷8 = 字节(B)，÷1024 = KB，÷1024÷1024 = MB

灰度直方图：

  • 描述各灰度值在图像中出现的次数/概率
  • 不能描述像素的空间关系
  • 一幅图像对应唯一直方图，但不同图像可有相同直方图（多对一）
  • 直方图均衡化能增强图像整体对比度

  • 连续性（相似性）：相邻像素灰度相似
  • 不连续性（突变性）：边缘处灰度突变

二、图像变换

知识点

傅里叶变换（DFT）特点：

  • 低频 → 图像变化平缓部分（主体）
  • 高频 → 图像边缘、噪声
  • 图像空域卷积 = 频域相乘（空域滤波 = 频域乘积运算）
  • 直流分量 F(0,0)F(0,0)F(0,0) = 图像灰度平均值
  • 二维DFT可由两次一维DFT实现（分离性）
  • 频谱中心化：乘以 (−1)x+y(-1)^{x+y}(−1)x+y
  • 相关性强的空间域像素 → 变换为不相关/弱相关的频域，反映频谱特性
  • 傅里叶变换具有能量集中作用

余弦变换（DCT）：

  • JPEG压缩中对图像子块做正交变换用离散余弦变换（DCT）
  • DCT系数用 Z字形（Zigzag） 重排，将AC系数按频率从低到高排列，增加"0"游程，提高压缩效率

沃尔什-哈达玛变换（WHT）：

  • 变换核由 +1、-1 组成的正交方阵
  • 哈达玛变换：行的列率排列无规则
  • 沃尔什变换：列率由小到大排列（二者区别即此）

图像变换的目的：

  • 使图像处理问题简化
  • 有利于图像特征提取
  • 有助于从概念上增强对图像信息的理解

三、图像压缩编码

知识点

三种数据冗余：

  1. 像素间冗余（空间相关性）
  2. 编码冗余（编码效率不足）
  3. 心理视觉冗余（人眼不敏感的信息）

无损压缩方法： Huffman编码、香农-费诺编码、算术编码、行程编码（RLE）

Huffman编码步骤：

  1. 将符号按概率升序排列
  2. 取最小两个概率相加，合成新概率
  3. 重复直到只剩一个概率为1（建立二叉树）
  4. 从根节点按前缀码规则进行0/1编码

评价指标：

Lˉ=∑ipi⋅li（平均码长）\bar{L} = \sum_i p_i \cdot l_i \quad \text{（平均码长）} Lˉ=i∑​pi​⋅li​（平均码长）

Cr=H原Lˉ（压缩比，H原为固定码长）C_r = \frac{H_{原}}{\bar{L}} \quad \text{（压缩比，}H_{原}\text{为固定码长）} Cr​=LˉH原​​（压缩比，H原​为固定码长）

JPEG压缩流程：

  • 图像分成8×8子块 → DCT → 量化 → Zigzag扫描 → 行程编码 + Huffman编码
  • 马赛克现象（块效应）原因：以8×8子块为单位解码，块间误差可见

四、图像增强

知识点

增强目的： 改善图像视觉效果，提高清晰度；便于后期特征分析。

直方图均衡化

公式：

sk=INT[(L−1)⋅∑j=0kp(rj)+0.5]s_k = \text{INT}\left[(L-1)\cdot\sum_{j=0}^{k}p(r_j) + 0.5\right] sk​=INT[(L−1)⋅j=0∑k​p(rj​)+0.5]

步骤：

  1. 统计各灰度级像素数 nkn_knk​
  2. 计算概率 p(rk)=nk/Np(r_k) = n_k / Np(rk​)=nk​/N
  3. 计算累积概率 Sk=∑j=0kp(rj)S_k = \sum_{j=0}^{k}p(r_j)Sk​=∑j=0k​p(rj​)
  4. 计算变换后灰度值（乘(L-1)后取整）
  5. 统计变换后直方图

平滑（低通滤波）

方法 特点
邻域平均 简单，会模糊边缘
中值滤波 非线性，保持边缘，去椒盐噪声效果好

中值滤波详细特点：

  • 对椒盐噪声（脉冲噪声）抑制效果明显
  • 不影响阶跃、斜坡信号
  • 连续个数小于窗口宽度一半的离散脉冲会被滤除
  • 会将点、尖峰滤除（细节丰富的图像不适合用中值滤波）

锐化（高通滤波）

  • 梯度算子（一阶微分）：Roberts、Prewitt、Sobel
  • 拉普拉斯算子（二阶微分）：各向同性，增强边缘

模板类型速判：

模板特征 类型
系数全正，和为1 平滑模板（低通）
系数有正负，和为0，中心为正 锐化/边缘检测模板（高通）
中心为4，上下左右为-1，其余0 拉普拉斯（4邻域）
中心为8，周围8个全为-1 拉普拉斯（8邻域）
左列负右列正（或反） Sobel/Prewitt梯度算子

知识点

退化原因： 散焦、运动模糊、大气湍流、噪声、成像系统非线性、底片失真等

退化模型：

g(x,y)=H[f(x,y)]+n(x,y)（连续）g(x,y) = H[f(x,y)] + n(x,y) \quad \text{（连续）} g(x,y)=H[f(x,y)]+n(x,y)（连续）

g=Hf+n（离散）g = Hf + n \quad \text{（离散）} g=Hf+n（离散）

退化函数估计方法：

  1. 图像观察法（凭经验观察）
  2. 实验估计法（用类似设备成像估计）
  3. 建模估计法（根据物理知识建数学模型）

复原方法：

方法 特点
逆滤波 直接用退化函数的逆滤波，噪声大时效果差
维纳滤波 需要知道图像信号和噪声的功率谱，考虑噪声，效果最好
最小二乘滤波 平滑度约束，噪声未知时使用

六、图像分割

知识点

三种分割方法对比：

方法 依据 适用场景
阈值分割 像素灰度的不连续性（直方图双峰） 目标与背景灰度差异大
边缘检测 像素灰度的不连续性（局部突变） 边缘轮廓提取
区域生长 像素灰度的连续性 确定图像内部区域

边缘检测算子对比：

算子 阶数 特点
Roberts 一阶 计算量小，定位精确，对噪声敏感
Prewitt 一阶 方向性好，抗噪中等
Sobel 一阶 方向性好，抗噪较好，最常用
Laplacian 二阶 各向同性，对噪声敏感，产生双边缘
LoG（高斯-拉普拉斯） 二阶 先平滑后检测，抗噪好
Canny — 计算量大，综合性能最好，多阈值跟踪

霍夫（Hough）变换：

  • 利用图像全局特性检测直线（或曲线）
  • 不用 y=kx+by=kx+by=kx+b 原因：近垂直直线斜率趋于无穷
  • 使用极坐标：ρ=xcos⁡θ+ysin⁡θ\rho = x\cos\theta + y\sin\thetaρ=xcosθ+ysinθ
  • 原理：图像空间中直线 → 参数空间中的点（点-线对偶性）

区域生长法： 从种子像素出发，按生长准则（如灰度差 ≤ T）将邻域像素合并，直到无新像素满足条件

边缘跟踪（双阈值法）：

  • 高阈值：检测明显边缘点
  • 跟踪阈值：高阈值点邻域内与其灰度差 ≤ 跟踪阈值的点被连接

对应题目

【试卷A·问答1】 简述阈值分割技术、边缘检测、区域生长法对图像进行分割依据的原理。

答：

  • 阈值分割：图像目标和背景具有不同灰度集合，用阈值T将目标与背景分开，依据像素灰度的不连续性
  • 边缘检测：基于图像灰度的不连续性，检测边缘作为分割依据，用差分、梯度、拉普拉斯算子等方法
  • 区域生长：基于图像灰度的连续性，从种子像素出发，将满足相似准则的邻域像素合并，确定内部区域

【试卷B·问答4】 简述霍夫（Hough）变换的基本原理。

答：

  1. 利用点-线对偶性：图像空间中共线的点，对应参数空间中共点的线
  2. 将图像空间中直线检测问题转化为参数空间中点的检测问题
  3. 参数空间中通过简单的累加统计完成点的检测
  4. 使用极坐标 ρ=xcos⁡θ+ysin⁡θ\rho = x\cos\theta + y\sin\thetaρ=xcosθ+ysinθ 避免斜率趋于无穷的问题

【试卷B·问答6】 请简述边缘检测的理论依据是什么，有哪些方法？各有什么特点？

答：

理论依据： 不同区域间灰度值的不连续性

方法与特点：

算子类型 代表算子 特点
一阶梯度 Roberts、Prewitt、Sobel 计算量小，方向性好，抗噪能力较差
二阶微分 Laplacian 各向同性，对噪声敏感，有双边缘
平滑+二阶 LoG（高斯拉普拉斯） 抗噪效果好，计算量较大
综合 Canny 综合性能最好，计算量大，多阈值跟踪

【试卷A·应用3】 边缘跟踪（检测阈值8，跟踪阈值2）

答：

  • （1）检测阈值处理：值 ≥8 的点被检出，标记为1，其余为0
  • （2）跟踪阈值处理：在已检出点基础上，对其邻域中值 ≥8 且与已检出点灰度差 ≤2 的点进行连接，形成连续曲线

七、形态学图像处理

知识点

四种基本运算：

运算 顺序 作用
腐蚀 — 收缩目标，去除小突起
