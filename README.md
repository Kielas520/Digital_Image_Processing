# Digital Image Processing (数字图像处理) 复习笔记与工具库

## 📚 项目简介
本项目旨在利用 `opencode` 和自定义的 `skills`，辅助进行《数字图像处理》(Digital Image Processing, DIP) 课程的期末复习与备考。项目不仅包含了系统的知识点总结、习题与错题，还配备了自动化脚本和 AI 辅助技能，以提高复习效率。

## 📂 目录结构

*   **`知识脉络/`**: 核心知识点整理、公式推导、概念梳理与思维导图。
*   **`侧重点/`**: 考试重点、重难点分析及考纲解析。
*   **`题集/`**: 课后练习题、历年真题及模拟题库。
*   **`错题集/`**: 记录做错的题目及详细解析，便于考前查漏补缺。
*   **`source/`**: 原始复习资源库，包含：
    *   `课件/`: 课程 PPT 或 PDF。
    *   `复习资料/`: 老师提供的复习大纲、讲义等。
    *   `实验答案/`: 课程实验相关资料。
    *   `extracted/`: 由脚本提取处理后的文本/图片等中间文件。
*   **`script/`**: 用于自动化处理文档的 Python 脚本。例如 `extract_all.py`、`extract_office.py` 等，可从 PPT 或文档中快速提取内容以便进行 AI 分析。
*   **`.opencode/`**: 包含针对本课程复习定制的 Opencode AI 技能 (`dip-review`)。

## 🛠️ 如何使用

1.  **资料处理**: 将新的 PPT 或文档放入 `source/` 中，运行 `script/` 下的脚本自动提取文本和图片。
2.  **系统复习**: 按照 `知识脉络/` 和 `侧重点/` 中的总结按章节复习（如：图像增强、图像复原、形态学等）。
3.  **刷题与巩固**: 在 `题集/` 中练习，并将易混淆概念和错题整理到 `错题集/` 中。
4.  **🤖 唤醒 AI 助教**: 本项目内置了 `dip-review` skill。你可以直接在命令行呼叫 AI：
    *   *"帮我总结一下直方图均衡化的考点"*
    *   *"从我的错题集里抽两道题考考我"*
    *   *"解释一下傅里叶变换在图像压缩中的应用"*
