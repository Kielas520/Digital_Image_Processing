# 人类视觉特性与颜色模型：生物学根源与色度学定义

为了确保在选择和填空题中拿满分，本篇文档对人眼光学解剖结构、心理视觉效应、以及红绿蓝三基色的标准波长与模型公式进行100%无死角梳理。

## 一、 人眼光学结构与视觉成像细胞 (Anatomy and Photo-receptors)

当光线射入眼睛时，通过以下光学系统在视网膜上成像：

### 1. 核心光学部件及其常考功能
- **角膜 (Cornea)**：光线进入眼睛的第一道半透明防护门。
- **晶状体 (Lens)**：相当于相机的可变焦镜头，通过其形变将外部光线精确聚焦在视网膜上。
- **虹膜 (Iris)**：其上的**瞳孔 (Pupil)** 相当于光圈。在强光下，瞳孔自动收缩以降低光强和视觉灵敏度，防止过载；在暗光下，瞳孔放大以收集更多光线。
- **视网膜 (Retina)**：相当于相机的感光 CMOS 传感器，视觉成像的最底层。
- **中央凹 (Fovea)**：视网膜的中心凹陷区。这里**高度密集分布了视锥（锥状）细胞，是人眼产生最清晰视觉（高分辨率、彩色）的地方**。
- **中央凹周围**：视觉分辨率急剧下降，视锥细胞急剧减少，**视杆（杆状）细胞急剧增多**。

### 2. 双感光细胞系统深度对比
- **视锥细胞 (Cones) —— 明视觉/彩色系统**：
  - *数量*：约 600万 ~ 700万。
  - *功能*：**负责彩色视觉、强光视觉（明视觉）、以及分辨物体的表面细节和精确轮廓（高空间分辨率）**。
  - *光谱分类*：分为蓝敏视锥细胞（敏感峰值 **$445\text{nm}$**）、绿敏视锥细胞（敏感峰值 **$535\text{nm}$**）和红敏视锥细胞（敏感峰值 **$575\text{nm}$**）。
  - *典型动物*：**鸡只有视锥细胞**，所以一到晚上就彻底看不清（夜盲）。
- **视杆细胞 (Rods) —— 暗视觉/黑白系统**：
  - *数量*：约 7500万 ~ 1.5亿（远多于视锥）。
  - *功能*：**负责黑白视觉（没有彩色信息）、对低亮度光极度敏感（暗视觉/夜视觉）、以及对运动物体的感知**。
  - *典型动物*：**猫头鹰只有视杆细胞**，能在漆黑夜里捕鼠，但看世界是黑白的。

## 二、 心理视觉效应：人眼的“自我欺骗”

### 1. 亮度适应 (Brightness Adaptation) 与 灰度分辨率
- **原理**：人眼绝对无法同时在整个亮度适应范围工作。它是通过改变自身的“亮度适应级”来逐步适应白天到夜晚的剧烈明暗变动。
- **人眼的灰度分辨能力**：与整个宽广的亮度适应范围相比，人眼能同时鉴别的光强度级总范围其实非常小，**人眼的主观灰度分辨能力只有 64 个灰度级左右**（超过 64 级渐变，人眼就难以分辨其亮度差，需要用计算机量化）。

### 2. 同时对比度 (Simultaneous Contrast)
- **定义**：人眼对某个区域感受到的主观亮度（主观感受），不仅依赖于它自身的物理亮度，还强烈地**受到它背景亮度的影响**。
- **规律**：同样的灰色方块，放在较暗（黑）背景里显得比较亮，放在较亮（白）背景里显得比较暗。

### 3. 马赫带效应 (Mach Bands)
- **定义**：在灰度有跳跃变化的边界处，人眼的主观视觉会**在边缘产生虚幻的明亮或黑暗条纹**。
- **物理意义**：这是人类视觉系统为了提高生存概率，**自带的一种增强物体边缘和轮廓对比度的生理反馈机制**。

## 三、 彩色视觉与颜色模型 (Color Models)

### 1. 加色三基色 (RGB) 的物理定义与色光匹配
根据色觉三基色原理，自然界中绝大部分可见光谱都可以用红(R)、绿(G)、蓝(B)三基色光按不同比例和强度混合表示：
- **标准基色波长（必考填空）**：
  - **红 (R)**：波长 $\lambda = \mathbf{700.0nm}$
  - **绿 (G)**：波长 $\lambda = \mathbf{546.1nm}$
  - **蓝 (B)**：波长 $\lambda = \mathbf{435.8nm}$
- **色光匹配方程 (Color Matching Equation)**：
  $$ C \equiv rR + gG + bB $$
  *(其中，$\equiv$ 代表视觉匹配， $r, g, b$ 代表三原色的比例系数，且满足：$r + g + b = 1$。)*

### 2. RGB 彩色图像的数字表示
- **原理**：RGB 模型是典型的**相加混色模型（加色模型）**。
- **强度表示**：每个像素的 R, G, B 分量分别分配一个 $[0, 255]$ 范围的强度值（各占 8 位）：
  - 纯红：$(255, 0, 0)$；纯黑：$(0, 0, 0)$；纯白：$(255, 255, 255)$；黄色：红+绿 $\rightarrow (255, 255, 0)$。
- **数据结构**：RGB 彩色图不能用一个单一矩阵描述，必须用**三个独立的基色平面矩阵**（红色通道、绿色通道、蓝色通道）共同描述。
- **像素深度与位面数量**：
  - RGB 真彩色像素包含 $8 \times 3 = \mathbf{24 \text{ bit}}$ 颜色信息。能生成 $2^{24} = 16,581,375$ 种颜色。
  - **位面数量 (Bit-plane quantity)**：一幅图像的位面数量相当于组成图像的像素矩阵维数。
    - **灰度图像是一个位面**（1个矩阵数据区）；
    - **彩色图像有三个位面**（红分量、蓝分量、绿分量 3个矩阵通道）。

### 3. 索引图像 (Indexed Images) 与 调色板 (Palette)
- **结构**：由一个**数据矩阵（存颜色索引号）**和一个**调色板（存实际的RGB色表）**组成。
- **特点**：像素点的值不存复杂的24位RGB，而是存对应调色板的列号（0~255）。极大地节省了存储空间（索引色图像占用磁盘空间很小，但质量不高），适用于多媒体动画和早期网页制作。
