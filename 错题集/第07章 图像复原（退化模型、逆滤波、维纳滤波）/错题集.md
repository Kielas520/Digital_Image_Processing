# 第07章 错题集

## 第07章 图像复原（退化模型、逆滤波、维纳滤波）

- **题目**：【单选】在进行图像逆滤波复原时，一旦图像中存在高频噪声干扰，复原结果往往会失效。其最根本的原因是：______。（选项：A. 因为高频处的传递函数 H(u, v) 往往会衰减到接近 0; B. 因为高频处的噪声 N(u, v) 强度恒等于 0; C. 因为逆滤波是一种有约束条件下的迭代复原技术; D. 因为逆滤波强行对图像进行了各向同性的二阶差分运算）
  - **用户答案**：D
  - **正确答案**：A
  - **错因**：对逆滤波噪声放大的“分母变小导致比值剧增”的代数机理理解不够直观，误将其与二阶差分拉普拉斯运算（D选项）联系在一起。
